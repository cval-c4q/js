Desk calculator
===============

Author: <carlovalenti@ac.c4q.nyc>


Polynomial expression evaluator and REPL interface

Currently evaluates arithmetical expressions, with explicit parenthesizing
Operators recongnized: +, -, *, /, %, ^ (exponentiation)
